1. Интеграция СКУД → СВН через ONVIF Profile C:
   - Передача событий доступа и тревог.
   - Активация записи камеры.
   - Маркировка видеопотока ID сотрудника.

2. Интеграция СВН → СКУД через REST API:
   - Передача события нарушения СИЗ.
   - Ограничение дальнейшего доступа.

3. Интеграция ЭКО → СКУД + СВН через Modbus/TCP:
   - Сигнал утечки газа.
   - Блокировка дверей.
   - Включение тревожной видеозаписи.

4. Шифрование:
   - HTTPS/TLS 1.3
   - AES-128 (OSDP Secure)
   - SSH для сервисных каналов.
